# Minimono Synthesizer

A high-performance, boutique-style virtual analog synthesizer inspired by the legendary 1970s hardware. Built with React and the Web Audio API.

![Minimono Overlay](https://github.com/user-attachments/assets/...) *Add a screenshot later*

## Features

### 🎹 Performance Ready
- **Computer Keyboard Support**: Play via home row (Z/X for octaves).
- **Smooth Pitch Bend & Mod**: Expression wheels with smooth hardware-style interpolation.
- **WebMIDI Integration**: Full support for external MIDI controllers at startup.

### 🔊 Boutique Audio Engine
- **Triple Oscillator Bank**: Custom waveforms (Tri, Tri-Saw, Rev-Saw, Saw, Sq, Pulse).
- **Classic Ladder Filter**: 24dB/oct resonant filter with keyboard tracking and contour controls.
- **Saturation Stage**: Soft/Hard clipping and West-Coast style **Wave Folding**.
- **Dual ADSR Envelopes**: Dedicated contours for loudness and filter.

### 💾 Patch Management
- **Preset Browser**: Load factory patches or save your own to local storage.
- **Export/Import**: Share your patches as `.json` files.
- **Randomizer**: Intelligently generate musically-relevant patches at the click of a button.

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`
4. Build for production: `npm run build`
